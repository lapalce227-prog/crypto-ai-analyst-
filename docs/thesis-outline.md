# 毕业论文大纲

## 题目
《基于大语言模型的加密货币交易行为分析平台的设计与实现》
Design and Implementation of a Cryptocurrency Trading Behavior Analysis Platform Based on Large Language Models

## 摘要（300字）
本文设计并实现了一个基于大语言模型的加密货币交易行为分析平台。平台支持交易记录的自动录入与统计，利用LLM对用户的交易行为进行智能复盘分析，并构建风险仪表盘可视化情绪、时段与盈亏的关联关系。系统采用前后端分离架构，前端基于React框架，后端基于FastAPI，AI模块对接DeepSeek大语言模型。平台旨在帮助交易者客观审视自身交易行为，降低情绪化决策带来的损失。测试结果表明，系统能够有效识别用户的亏损模式并生成有价值的复盘建议。

关键词：大语言模型；交易行为分析；加密货币；复盘系统；风险可视化

---

## 第一章 绪论（2500字）

### 1.1 研究背景
- 加密货币市场的高速发展与散户参与
- 散户交易者普遍存在的情绪化交易问题
- AI技术在金融分析领域的应用趋势

### 1.2 国内外研究现状
- 国外：TokenMetrics的AI评分系统、Nansen的链上数据智能分析、DefiLlama的对话式查询
- 国内：Bitget GetAgent、币安Ai Pro等交易所内置AI工具
- 现有工具的不足：面向专业用户、价格高昂、缺乏中文支持、缺少行为复盘功能

### 1.3 研究目的与意义
- 面向普通散户提供低成本、易用的交易行为分析工具
- 利用LLM的自然语言能力降低使用门槛
- 通过行为数据分析帮助用户建立理性交易习惯
